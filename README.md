## proyectos propios

### 🪢 Quipu
App de finanzas personales pensada para el mercado peruano. Usa el método de sobres 50/30/20 y tiene soporte para trabajadores independientes (ingresos variables, sin quincena fija).
> *Next.js · Convex · TypeScript*
> *https://quipu-finance.app*

### 🎨 Inkyra
Inkyra es una aplicación que ayuda a tatuadores a automatizar la fase inicial de ideación y cotización de tatuajes usando inteligencia artificial. Los usuarios describen su idea mediante un formulario guiado y la app genera un diseño conceptual con IA; si el resultado les gusta, pueden enviarlo directamente al tatuador junto con las especificaciones del tatuaje para recibir una cotización más rápida y precisa.

> *Nextjs · PostgresQL · Typescript*
> *https://inkyra.app*
